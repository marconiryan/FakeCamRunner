### Instalação
```bash
sudo apt update
```

```bash
sudo apt install openjdk-17-jdk
```

### Como usar

```bash
java -jar app.jar -u <URL da sala> -b <Quantidade de Bots (padrao 3)>
```


### Exemplo

```bash
java -jar app.jar -u https://example.com -b 3
```
